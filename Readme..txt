Esta pasta e subpastas contém dados e scripts para a ferramenta de Dissecação Vertical. 
- Para executar a ferramenta abra uma interface do ArcCatalog e acesse a toolbox "Dissecação Vertical.tbx" e clique na ferramenta "Dissecação Vertical". 
- Para informações sobre o funcionamento da ferramenta e dados espaciais necessários consulte a subpasta "Doc" e a aba lateral da ferramenta.
- O arquivo "Dissecação Vertical - Exemplo.mxd" apresenta exemplo de dados espaciais que são necessários para executar a ferramenta e o resultado alcançados com o seu processamento. 
- A pasta "Scratch" esta vazia e pode ser utilizada pelo usuário como pasta para armazenar os arquivos intermediários do processamento.

Avisos:
Esta ferramenta foi desenvolvida e testada na versão 10.2 do software ArcGIS.
Todos os dados vetoriais de entrada devem estar localizados dentro de um banco de dados. Não utilizar arquivos no formato Shapefile.
